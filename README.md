# 🤖 Python Personal Assistant

A simple command-line Personal Assistant built using Python. It can perform everyday tasks like greeting users, telling the current time and date, opening websites, launching applications, and responding to basic commands.

## 🚀 Features

- 👋 Greets the user
- 🕒 Displays current time
- 📅 Shows current date
- 🌐 Opens websites (Google, YouTube, etc.)
- 📂 Opens applications
- 💬 Responds to simple text commands
- ❓ Help menu for available commands

## 🛠️ Technologies Used

- Python 3
- datetime module
- os module
- webbrowser module
