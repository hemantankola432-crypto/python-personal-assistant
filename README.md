# 🤖 Python Personal Assistant

## 📌 Overview

The Python Personal Assistant is a command-line application developed using Python to perform everyday tasks through simple text-based commands. It acts as a virtual assistant capable of responding to user requests such as displaying the current date and time, opening websites, launching applications, and providing helpful responses.

This project demonstrates the practical use of Python programming concepts and standard libraries to automate routine tasks. It is an excellent beginner-friendly project that showcases problem-solving, logical thinking, and software development skills.

---

## ✨ Features

- 👋 Greets the user with friendly messages.
- 🕒 Displays the current system time.
- 📅 Shows the current date.
- 🌐 Opens popular websites such as Google and YouTube.
- 💻 Launches applications from the computer.
- 💬 Responds to user commands.
- ❓ Displays available commands through a help menu.
- 🚪 Safely exits the application.

## ▶️ How to Run

### Prerequisites

- Python 3.x installed on your system.

### Steps

1. Clone this repository.

```bash
git clone https://github.com/your-username/python-personal-assistant.git
```

2. Navigate to the project folder.

```bash
cd python-personal-assistant
```

3. Run the program.

```bash
python assistant.py
```
